# 广州市谷麦夏餐饮管理有限公司 - 官方网站

广州市谷麦夏餐饮管理有限公司官网项目，旨在展示公司餐饮品牌加盟、运营管理及供应链服务。

## 🚀 项目特性

- **模块化架构**：CSS 和 JS 按页面拆分，确保样式隔离，易于维护。
- **响应式设计**：完美适配桌面端、平板及移动端设备。
- **动态交互**：首页包含高性能轮播图，各页面均有平滑的过渡效果。
- **SEO 友好**：清晰的 HTML 结构，语义化标签。

## 📁 项目目录结构

```text
gumaixia-official/
├── index.html              # 网站首页
├── about.html              # 关于我们
├── services.html           # 核心业务
├── contact.html            # 联系我们
├── news/                   # 新闻动态模块
│   ├── index.html          # 新闻列表页
│   ├── 2026-01-new-service.html # 新闻详情：智能厨房解决方案
│   └── 2026-02-event.html       # 新闻详情：参加餐饮加盟展
├── assets/                 # 静态资源
│   ├── css/                # 样式表
│   │   ├── reset.css       # 样式重置
│   │   ├── main.css        # 全局公共样式
│   │   ├── index.css       # 首页特定样式
│   │   ├── about.css       # 关于我们特定样式
│   │   ├── services.css    # 核心业务特定样式
│   │   ├── news.css        # 新闻模块特定样式
│   │   └── contact.css     # 联系我们特定样式
│   ├── js/                 # 脚本
│   │   ├── common.js       # 全局公共脚本（导航、移动端菜单）
│   │   └── index.js        # 首页特定脚本（轮播图逻辑）
│   └── images/             # 图片资源
├── .gitignore              # Git 忽略文件配置
├── .gitattributes          # Git 属性配置
└── README.md               # 项目说明文档
```

## 🛠️ 技术栈

- **HTML5**：语义化页面结构
- **CSS3**：Flexbox & Grid 布局，自定义变量
- **JavaScript (Vanilla)**：原生 JS 实现动态交互
- **Font Awesome**：图标库支持

## 💻 本地开发

1. 克隆项目：
   ```bash
   git clone https://github.com/your-repo/gumaixia-official.git
   ```
2. 直接在浏览器中打开 `index.html` 即可预览效果。

## 📄 维护说明

- **样式修改**：全局样式请修改 `main.css`，页面特定样式请在对应的 `.css` 文件中修改。
- **脚本修改**：全局交互（如导航栏）位于 `common.js`。
